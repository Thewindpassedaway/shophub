## ✅ 第五步: 连接服务器 (预计2分钟)

### Windows用户 - 使用PuTTY

#### 转换密钥格式
- [ ] 下载 PuTTY: https://www.putty.org/
- [ ] 运行 `puttygen.exe`
- [ ] 点击 **Load**,选择 `.pem` 文件
- [ ] 点击 **Save private key**,保存为 `.ppk`

#### 连接服务器
- [ ] 运行 `putty.exe`
- [ ] Host Name: `opc@你的服务器IP`
- [ ] Port: `22`
- [ ] 左侧: **Connection** → **SSH** → **Auth**
- [ ] 选择 `.ppk` 文件
- [ ] 点击 **Open**
- [ ] 首次连接输入 `yes`

### Mac/Linux用户
- [ ] 执行: `chmod 400 oracle-key.pem`
- [ ] 执行: `ssh -i oracle-key.pem opc@你的服务器IP`

## 🔧 常用命令速查

```bash
# 查看应用状态
pm2 status

# 查看日志
pm2 logs shophub

# 重启应用
pm2 restart shophub

# 停止应用
pm2 stop shophub

# 启动应用
pm2 start shophub

# 备份数据库
mysqldump -u root -p shophub > backup_$(date +%Y%m%d).sql

# 更新代码
cd ~/merchant-collection
git pull
pm2 restart shophub
```

## ❓ 遇到问题?

### 无法连接服务器?
- [ ] 检查防火墙是否开放22端口
- [ ] 检查私钥文件是否正确
- [ ] 检查IP地址是否正确

### 应用无法访问?
- [ ] 检查防火墙是否开放3000端口
- [ ] 检查应用状态: `pm2 status`
- [ ] 查看日志: `pm2 logs shophub`

### 数据库连接失败?
- [ ] 检查MySQL是否运行: `sudo systemctl status mysqld`
- [ ] 检查`.env`文件配置
- [ ] 检查数据库密码是否正确
