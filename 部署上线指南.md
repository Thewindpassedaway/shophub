## 🎯 方案一：部署到Railway（推荐）⭐⭐⭐⭐⭐

### 优势
- ✅ 免费额度充足
- ✅ 自动HTTPS
- ✅ 内置MySQL数据库
- ✅ 自动部署（连接GitHub）
- ✅ 国外服务器，无需备案

### 步骤1：准备GitHub仓库

```bash
# 1. 初始化Git（如果还没有）
git init

# 2. 添加所有文件
git add .

# 3. 提交
git commit -m "Initial commit: ShopHub PWA应用"

# 4. 在GitHub创建新仓库
# 访问 https://github.com/new
# 创建名为 shophub 的仓库

# 5. 关联远程仓库
git remote add origin https://github.com/Thewindpassedaway/shophub.git

# 6. 推送代码
git push -u origin main
```

### 步骤2：注册Railway

1. **访问Railway**
   - 网址：https://railway.app
   - 点击"Login" → 选择"Login with GitHub"

2. **授权GitHub**
   - 允许Railway访问你的仓库

### 步骤3：创建新项目

1. **点击"New Project"**

2. **选择"Deploy from GitHub repo"**
   - 选择你的 `shophub` 仓库
   - 点击"Deploy Now"

3. **Railway会自动识别配置**
   - 检测到 `railway.json` 和 `Procfile`
   - 自动配置Node.js环境

### 步骤4：添加MySQL数据库

1. **在项目页面点击"+ New"**

2. **选择"Database" → "Add MySQL"**

3. **等待数据库创建完成**
   - Railway会自动 provision 一个MySQL实例

4. **获取数据库连接信息**
   - 点击MySQL服务
   - 进入"Variables"标签
   - 复制以下变量：
     - `MYSQLHOST`
     - `MYSQLPORT`
     - `MYSQLUSER`
     - `MYSQLPASSWORD`
     - `MYSQLDATABASE`

### 步骤5：配置环境变量

1. **回到你的主服务（shophub）**

2. **进入"Variables"标签**

3. **添加以下环境变量**：

```
DATABASE_HOST = <从MySQL复制的MYSQLHOST>
DATABASE_PORT = <从MySQL复制的MYSQLPORT>
DATABASE_USER = <从MySQL复制的MYSQLUSER>
DATABASE_PASSWORD = <从MySQL复制的MYSQLPASSWORD>
DATABASE_NAME = <从MySQL复制的MYSQLDATABASE>
JWT_SECRET = your-secret-key-change-this
NODE_ENV = production
```

4. **点击"Add"保存每个变量**

### 步骤6：初始化数据库表

1. **在Railway项目页面**
   - 点击右上角"Shell"或"Console"

2. **执行数据库初始化脚本**：

```bash
# 进入backend目录
cd backend

# 运行初始化脚本（根据你的实际脚本名称调整）
node scripts/createShopChecksTable.js
```

或者在本地执行后推送：
```bash
# 本地连接Railway数据库执行SQL
mysql -h <host> -P <port> -u <user> -p<password> <database> < schema.sql
```

### 步骤7：等待部署完成

1. **查看部署日志**
   - 进入"Deployments"标签
   - 查看实时日志

2. **等待状态变为"Success"**
   - 通常需要2-5分钟

3. **获取访问地址**
   - 点击"Settings" → "Domains"
   - 复制生成的域名（如：`xxx.up.railway.app`）

### 步骤8：测试访问

1. **在浏览器打开生成的域名**
   - 应该能看到ShopHub登录页面

2. **测试功能**
   - 登录
   - 查看店铺
   - 提交检查记录

3. **测试PWA安装**
   - 用手机浏览器访问
   - 按照《手机端全屏安装指南》安装

## 🔧 部署后配置

### 1. 创建管理员账号

部署成功后，你需要创建第一个管理员账号：

```bash
# 在Railway Shell中执行
cd backend
node scripts/addAdmin.js
```

或者通过API创建：
```javascript
// 使用Postman或curl
POST https://your-domain.up.railway.app/api/auth/register
{
  "username": "admin",
  "password": "your-password",
  "displayName": "管理员",
  "role": "admin"
}
```

### 2. 导入初始数据

如果有初始店铺数据：
```bash
# 执行数据导入脚本
node scripts/importShops.js
```

### 3. 配置自定义域名（可选）

1. **在Railway中**
   - Settings → Domains
   - 点击"Add Custom Domain"
   - 输入你的域名

2. **配置DNS**
   - 添加CNAME记录指向Railway提供的地址

## 🆘 常见问题

### Q1: 部署失败怎么办？
**检查**：
- 查看Railway部署日志
- 确认环境变量配置正确
- 检查数据库连接信息

### Q2: 数据库连接失败？
**解决**：
- 确认MySQL服务已创建
- 检查环境变量是否正确
- 确认数据库表已创建

### Q3: 图片上传失败？
**检查**：
- 确认uploads目录权限
- 检查磁盘空间
- 查看错误日志

### Q4: PWA无法安装？
**确认**：
- 网址必须是HTTPS
- manifest文件正确加载
- Service Worker注册成功
